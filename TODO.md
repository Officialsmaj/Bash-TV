# Bash TV Media - Frontend News Website Project

## Project Overview
Create a professional Hausa news website frontend called "Bash TV Media" using HTML5, CSS3, and Vanilla JavaScript.

### Phase 2: CSS Files

### Phase 3: JavaScript Files

- [ ] Verify all files are created

- [ ] Verify all JavaScript functionality
